Il progetto consiste in una pipeline bioinformatica sviluppata in ambiente MATLAB per l’analisi e la modellazione del gene dell'insulina umana (NM_000618). Il software automatizza l'intero flusso di lavoro: dall'acquisizione dati in tempo reale tramite API REST (NCBI E-Utilities), alla generazione di un modello geometrico 3D della doppia elica basato sulla complementarità di Watson-Crick, fino alla gestione persistente dei dati tramite database relazionali SQLite e interrogazioni in linguaggio SQL.
